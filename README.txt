Как да тестваш:
1) Качи файловете в GitHub Pages (както при предишното приложение).
2) Отвори линка на телефона (Chrome) → „Добавяне към началния екран“.
3) Добави 1–2 МПС, експортирай .ics и добави в Google Calendar за реални аларми.

За истински фон-напомняния без календар → следва APK версия (Capacitor/Android Studio).
